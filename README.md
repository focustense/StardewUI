## Installation

StardewUITest follows the standard installation procedure for most mods:

1. Install SMAPI and set up your game for mods, per the [Modding: Player's Guide](https://stardewvalleywiki.com/Modding:Player_Guide/Getting_Started) instructions.
2. Download the [latest release](https://github.com/focustense/StardewUITest/releases). Make sure you download the `StardewUITest x.y.z.zip` file, **not** the source code.
3. Open the .zip file and extract the `StardewUITest` folder into your `Stardew Valley\Mods` folder.
4. Launch the game!
